# ⚡️ Team 404 Found

404 Not Found라는 개발자라면 누구나 한 번쯤 접했을 HTTP 표준 응답 코드로, 서버 자체는 존재하지만 서버에서 요청한 것을 찾을 수 없을 때 사용하는 에러 응답 코드입니다. <br>
이 중 Not Found의 Found 라는 반전적 의미를 따와, **“길을 잃고 막히는 순간에도 답을 찾는 개발자 팀‘** 이라는 메시지를 담아 **”404 Found“** 라는 이름을 지었습니다.
<br>
<br>

# 🌱 당신의 여정을 그려줄 단 하나의 로드맵, DevDo

**DevDo**는 개발자를 의미하는 **Dev**eloper 와 해야 할 일을 정리하고 관리하는 할 일 리스트를 의미하는 to**Do**의 합성어로, 개발자를 꿈꾸는 학생들이 학습 목표를 효과적으로 관리하고
성취할 수 있도록 돕기 위해 개발한 웹 서비스입니다. <br>
<br>
많은 학생들이 학습 계획을 세우고 목표를 관리하기 위해 Todo 어플이나 학습 플래너를 작성하지만, 개발이라는 분야에서 복잡하고 방대한 학습 내용으로 인해 계획을 세우는 것 자체에 어려움을 느끼곤 합니다. <br>
프로그래밍 학습은 일반적인 학업과 달리, 단순히 공부하는 것을 넘어 끊임없이 새로운 기술을 익히고 프로젝트를 통해 실력을 쌓아 나가야 하기 때문입니다. <br>
<br>
**DevDo**는 이러한 개발 학습의 특수성을 고려하여, 사용자가 단순히 할 일을 나열하는 것을 넘어 **체계적인 학습 로드맵을 제공**하고, 추가적으로 개발 분야가 아니더라도 목표하고자 하는 다른 분야에 적용하여 **계획을 세우고 목표를 달성**할 수 있도록 개발되었습니다.
<br>
<br>

# 🔥 주요 기능 (서비스 사용 흐름)

### 1️⃣ 로드맵 관리

사용자는 로드맵 관리 페이지에서 학습 로드맵을 생성, 수정, 삭제할 수 있습니다.<br>
또한 드래그 앤 드롭을 통해 편리하게 로드맵 목록을 관리 가능합니다.
<br>
<br>

### 2️⃣ 학습 로드맵

로드맵 관리 페이지에서 로드맵 생성 후 진입하는 페이지입니다.

DevDo 웹사이트에서 제공하는 기본 로드맵인 `Frontend 로드맵`, `Backend 로드맵`, `협업 로드맵`을 통해 각 분야별 학습 목표와 단계를 한 눈에 확인할 수 있습니다. <br>
또한 `유저 커스텀 로드맵`을 통해 개인 학습 계획을 자유롭게 설계하고, 자신만의 학습 경로 구성도 가능합니다.<br>
로드맵은 마인드맵 형식으로 구성되어 있으며, 노드 추가, 삭제, 드래그, 이름 변경, 컬러 변경 등의 기능을 통해 사용자 편의에 맞게 로드맵을 수정할 수 있습니다.
<br>
<br>

### 3️⃣ 학습 메모

로드맵 속 노드를 통해 이동할 수 있는 메모 페이지입니다.

한 노드 당 1개의 메모 페이지를 생성할 수 있으며,
블록 기반의 리치 텍스트 편집기를 이용해 체크리스트, 토글, 코드 블록 등의 기능을 제공하여 편리하게 학습 메모를 작성할 수 있습니다.<br>
또한 해당 페이지 내에서 AI 추천 기능을 통해 학습 자료 및 강의 연계, 학습 커리큘럼 추천 등이 가능합니다.
<br>
<br>

### 4️⃣ 커뮤니티

학습 과정에서 질문, 피드백, 정보 공유 등을 할 수 있는 커뮤니티 페이지입니다.

댓글, 좋아요를 통해 다른 사용자와 의사소통이 가능하며, 스크랩 기능으로 유용한 정보를 저장할 수 있습니다.<br> 스크랩한 글은 마이페이지에서 확인 가능합니다.<br>
또한 팔로우 / 팔로잉 기능을 통해 다른 사용자와의 네트워킹이 가능하며 상대방의 프로필 조회가 가능합니다.
<br>
<br>
